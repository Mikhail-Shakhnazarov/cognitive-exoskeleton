# How to Cite This Work

## Academic Citation

```
Shakhnazarov, M. (2025). Cognitive Exoskeleton Framework: Governance Infrastructure 
for LLM-Mediated Knowledge Work. Retrieved from 
https://github.com/Mikhail-Shakhnazarov/cognitive-exoskeleton
```

## BibTeX

```bibtex
@misc{shakhnazarov2025cognitive,
  author = {Shakhnazarov, Mikhail},
  title = {Cognitive Exoskeleton Framework: Governance Infrastructure for LLM-Mediated Knowledge Work},
  year = {2025},
  publisher = {GitHub},
  journal = {GitHub repository},
  howpublished = {\url{https://github.com/Mikhail-Shakhnazarov/cognitive-exoskeleton}},
  note = {Licensed under MIT (code) and CC BY 4.0 (documentation)}
}
```

## In-Text Reference

When referencing this work in blog posts, documentation, or informal writing:

> This approach builds on the Cognitive Exoskeleton framework (Shakhnazarov, 2025), 
> which provides governance infrastructure for LLM-mediated knowledge work.

Or simply:

> Using cognitive exoskeleton principles (https://github.com/Mikhail-Shakhnazarov/cognitive-exoskeleton)...

## Attribution in Derivative Works

If you build upon or adapt this work, please:

1. Include link to this repository
2. Mention original author: Mikhail Shakhnazarov
3. Note any significant modifications you've made
4. Maintain the CC BY 4.0 license for documentation you've adapted

Example:

> This project uses a modified version of the devOS implementation from 
> the Cognitive Exoskeleton framework by Mikhail Shakhnazarov 
> (https://github.com/Mikhail-Shakhnazarov/cognitive-exoskeleton).
> Modifications include [describe changes].
